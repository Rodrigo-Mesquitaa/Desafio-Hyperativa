# Teste-Hyperativa

Teste Web Application - API DotNet 5.0.Projeto Application de consulta arquivo TXT.

# Projeto

- Realize o donwload ou git clone abra o projeto no visual studio.
- Para inicialização do projeto F5 - ou Start.
- Para realizar o teste abrar o projeto Test - Debug Build para realização dos testes.

 
## Tecnologias utilizadas

- Visutal Studio 2019
- C# 
- DOTNET 5.0
- Architecture DDD.
- Swagger
- Authentication Jwt
- Injection Dependency
- Tets
- Git Clone
